使用GitHub免費發佈網頁

首先：註冊 GitHub 網站　

以下幾個步驟可以完成發佈：

第一步：準備好您要發佈的網頁，放在資料夾裡（可以放在桌面上）
第二步：在GitHub按下 New Repository
第三步：提供New Repository 名稱，必須遵守這個規則：名稱.github.io，完成後按下新增

第四步：按下uploading an existing file，然後將網頁資料夾移到 New Repository 裡

第五步：上傳檔案之後，在Commit 輸入訊息，最後再按下送出

第六步：到Settings / GitHub Pages 裡，按下Check it out here. 裡頭的Source選擇main 並Save

第七步：按下網址，並在網址後面加上/資料夾名稱/首頁名稱

【導言】
咖啡是一種令人沉醉的飲品，它的獨特風味和香氣能夠帶給人們無限的享受和滿足感。我們的網站旨在帶領您深入了解咖啡的世界，探索各個產地產區的風味特色，並提供關於咖啡沖泡的詳細指南和技巧。

【首頁】

    引人入勝的圖片和精心設計的介紹文字，展示咖啡的誘人魅力。
    簡單明瞭的導航欄，方便用戶流暢地尋找所需的資訊。

【咖啡產地產區】

    在這個部分，我們將介紹各個主要咖啡產地產區的風味特點和栽種方式。從南美洲的哥倫比亞到非洲的埃塞俄比亞，我們將帶您穿越世界各地的咖啡園，探索獨特的土壤和氣候如何影響咖啡豆的風味。

【咖啡沖泡指南】

    這一部分將提供詳細的咖啡沖泡指南和技巧，包括傳統的手沖、濾杯、法壓壺和咖啡機等各種沖泡方法。
    我們將解釋每種方法的原理，推薦適合的咖啡豆種類，並提供步驟清晰的圖文教學，以幫助您輕鬆地沖泡出美味的咖啡。
預備材料：
準備咖啡壺（例如濾杯、法壓壺）、濾紙或金屬濾網、磨好的咖啡豆、熱水壺和計時器。
加入咖啡豆：
將磨好的咖啡豆倒入濾紙或金屬濾網中，確保咖啡粉末均勻平整，並根據個人口味調整用量。
澆水沖泡：
熱水壺中煮沸水，稍等片刻待水溫降至約90°C，然後開始澆水沖泡。先倒入一些水，使咖啡舒展膨脹，再緩慢均勻地倒入剩餘的熱水，讓水與咖啡粉充分接觸。
等待沖泡：
等待約3-4分鐘的沖泡時間，讓咖啡粉充分浸泡並釋放出風味。
沖泡完成：
沖泡時間結束後，輕輕攪拌咖啡漿（表面的咖啡泡沫），再將手沖咖啡倒入咖啡杯中。現在，您可以享受濃郁香醇的手沖咖啡了！

    哥倫比亞：
    哥倫比亞的咖啡豆深受世界愛好者的喜愛。這些豆子具有平衡的口感和濃郁的巧克力、堅果風味，帶有微妙的甜味和柔和的酸度，讓您品味到來自南美的獨特風味。

    巴拿馬：
    巴拿馬的咖啡豆屬於稀有珍品，被譽為咖啡中的極品。它們通常呈現出清新、花果香氣，帶有蜜糖甜味和柔順的酸度，並帶有柑橘和紅醋栗等水果的風味，為您帶來極致的品嚐體驗。

    埃塞俄比亞西達摩的咖啡豆擁有驚艷的多樣風味。這些豆子常呈現出豐富的水果香氣，如藍莓和覆盆子，伴隨著清新的花香和茶葉般的香氣。口感柔順而光滑，帶有令人愉悅的酸度，讓您沉浸在一場芬芳的咖啡旅程。

    巴西（Brazil）：
    巴西咖啡以其濃郁豐富而受到喜愛。這些豆子帶有明顯的巧克力和堅果風味，伴隨著微妙的焦糖甜味。口感醇厚而圓潤，讓人回味無窮。作為世界上主要的咖啡生產國之一，巴西的咖啡絕對值得一試。

    曼特寧（Mandheling）：
    來自印度尼西亞的曼特寧咖啡以其獨特濃郁的風味而聞名。這些豆子具有濃郁的深色巧克力和煙燻的特點，伴隨著木質和土壤香氣。酸度較低，帶有濃厚的醇厚口感，成為喜愛濃烈口味的咖啡愛好者的最佳選擇。

    哥斯大黎加（Costa Rica）：
